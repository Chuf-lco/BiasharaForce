BiasharaForce 
A simple Python-based Agentic data extraction tool for Kenyan M-Pesa SMS messages. It uses the Groq API (Llama 3.1) to parse unstructured M-Pesa text into clean, structured JSON data.

How it works
It takes a raw M-Pesa confirmation message and uses an LLM to extract the amount, sender_name, sender_phone, and date, outputting strict JSON that can be fed into a database or another Agent.

It is now a full working prototype.
